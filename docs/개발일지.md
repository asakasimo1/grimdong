## 2026-03-13 | 버그 수정 3건

### 1. SettingsPage 모바일 빈화면

- **원인**: `data.photos = {}` (빈 JSONB) → optional chaining 미적용 → `photos.persons` undefined → TypeError
- **해결**: `setPhotos({ persons: data.photos?.persons ?? {}, places: data.photos?.places ?? {} })`

### 2. AI 변환 오류 (TransformModal)

- **원인**: Gemini 2.5 Flash thinking 토큰이 `maxOutputTokens: 120` 전부 소비
- **해결**: `thinkingConfig: { thinkingBudget: 0 }` 추가

### 3. Supabase Storage 한글 파일명 오류

- **원인**: Storage는 한글 경로 미지원
- **해결**: `KEY_ASCII` 매핑 테이블 (아이→child, 엄마→mom, 아빠→dad 등) + `toAscii()` 함수 적용
  - 적용 파일: `SettingsPage.jsx`, `PhotoRequestModal.jsx`
